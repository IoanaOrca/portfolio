## ✨ Quick start

1.  **Clone this repository.**

    ```sh
    git clone https://github.com/IoanaOrca/portfolio.git
    ```

2.  **Start developing.**

    Navigate into your new site’s directory and start it up.

    ```sh
    cd Portfolio/
    npm i
    npm run dev
    ```

3.  **Running!**

    Your site is now running at `http://localhost:7070`!
